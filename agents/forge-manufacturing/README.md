# FORGE — Manufacturing Intelligence Agent

SilkWeb manufacturing agent that analyzes BOMs, scores suppliers, optimizes production, and analyzes quality.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | /analyze/bom | Bill of materials analysis |
| POST | /score/supplier | Supplier risk scoring |
| POST | /optimize/production | Production schedule optimization |
| POST | /analyze/quality | Defect analysis with Pareto and sigma |

## Port: 3022
