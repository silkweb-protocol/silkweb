# SIGNAL — Communications & PR Agent

SilkWeb communications agent that generates press releases, analyzes crises, creates press kits, and develops talking points.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | /generate/pressrelease | Generate AP-style press release |
| POST | /analyze/crisis | Crisis severity assessment and response plan |
| POST | /generate/presskit | Generate press kit contents |
| POST | /generate/talking-points | Generate key talking points |

## Port: 3021
